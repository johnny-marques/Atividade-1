O que é o inferno das dependências? 
O inferno de dependências é uma designação dada a complicações no lidar com pacotes de software.

O que houve no caso do Jarbas? fale um pouco deste projeto.
Ao desenvolver uma nova versão do software, a versão 0.4.0, eles tiveram um problema gerado por um código externo, mas realmente para ter certeza disso eles voltaram uma versão do software, para a versão 0.3.0 que era estável e funcional, para que fosse possível verificar se o problema era realmente externo ou do código novo.
 
O que é o PyUP? 
É um programa que faz versionamento de arquivos em Python. 
Ele atualiza automaticamente as alterações feitas no código Python criando pull requests no Github contendo Changelogs descrevendo o que foi alterado. 
Além disso, ele efetua testes de segurança e falhas nas atualizações das dependências.

Como ele pode contribuir para o gerenciamento da configuração de software? 
Ele serve para gerenciar as alterações feitas nos códigos dos softwares.
